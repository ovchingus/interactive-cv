# Resume builder

This is a fast deployable, universal resume.

## How to deploy to GitHub Pages
 1) Change deploy repo information in `package.json`. Also you need to configure right path in localization options. To do this, change path in `src/i18n.js` file.

 2) Add personal information you want to include for resume in `public/locales/{{lang}}/translation.json`
